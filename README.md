# AI-Proposal-Quotation-Generator
An AI-powered internal tool for GWS Digital Service that automates the creation of client proposals, quotations, and outreach emails — turning a client requirement form into a polished, downloadable PDF proposal + quotation in a few clicks.

Built with Python + Streamlit + SQLite, with optional OpenAI or Gemini integration for richer AI-written content. Works fully offline out of the box using a built-in template engine — no API key required to run or demo it.
